E1 is {{ equation1 }} and E2 is {{ equation2 }}

TASK: Does E1 imply E2 over all magmas?
DEFAULT ANSWER: TRUE. Switch to FALSE only when a test below proves separation.

OUTPUT (first line must be the verdict):
VERDICT: TRUE or FALSE
REASONING: (one sentence)
PROOF: (brief)
COUNTEREXAMPLE: (if FALSE, name the test and show E1 holds / E2 fails; if TRUE, "None")

═══ QUICK TRUE ═══
If E1 has the form "x = T" and the variable x does NOT appear anywhere in T,
answer TRUE at once (singleton source rule).

═══ THE 4 TESTS ═══
For each test, decide independently: does E1 HOLD? Does E2 HOLD?
A separation is: E1 HOLDS and E2 FAILS on the SAME test → FALSE.
If none of the 4 tests separates → TRUE.

CRITICAL RULE — evaluate each equation ALONE:
  Each test compares the LEFT side of = against the RIGHT side of = inside
  ONE equation. Never compare a variable count or property BETWEEN E1 and E2.

When in doubt whether a test holds → say FAILS.
(Defaulting to FAILS prevents phantom separations and keeps the TRUE default safe.)

─── TEST 1: LP (first variable) ───
Find the FIRST letter (a–z) on the left side of =.
Find the FIRST letter (a–z) on the right side of =.
Same letter → HOLDS.  Different → FAILS.

  "x = (y * z) * x"  →  first(LHS)=x  first(RHS)=y  →  x≠y  →  FAILS
  "x = (x * y) * z"  →  first(LHS)=x  first(RHS)=x  →  x=x  →  HOLDS
  "x * y = (x * z) * w"  →  first(LHS)=x  first(RHS)=x  →  HOLDS

─── TEST 2: RP (last variable) ───
Find the LAST letter on the left side of =.
Find the LAST letter on the right side of =.
Same letter → HOLDS.  Different → FAILS.

  "x = y * (z * x)"  →  last(LHS)=x  last(RHS)=x  →  HOLDS
  "x = (x * y) * z"  →  last(LHS)=x  last(RHS)=z  →  x≠z  →  FAILS
  "x * y = z * (w * y)"  →  last(LHS)=y  last(RHS)=y  →  HOLDS

─── TEST 3: C0 (star-presence) ───
Check whether each side of = contains at least one * symbol.

  Both sides have *  →  HOLDS
  Neither has * and same bare variable  →  HOLDS
  One side has * and the other is a bare variable  →  FAILS
  Neither has * but different variables  →  FAILS

  "x * y = (z * w) * x"  →  both have *  →  HOLDS
  "x = (y * z) * w"  →  LHS bare, RHS has *  →  FAILS
  "x = x"  →  neither has *, same var  →  HOLDS

─── TEST 4: VARS (variable-set match) ───
List every distinct letter on the left side → set L.
List every distinct letter on the right side → set R.

  If both sides have *:  L = R → HOLDS.  L ≠ R → FAILS.
  If neither has * and same variable:  HOLDS.
  If one side is a bare variable v (no *) and the other has *:
      HOLDS only when the * side uses no variable other than v.
  Otherwise: FAILS.

  "x * y = (y * x) * y"  →  L={x,y}  R={x,y}  →  same  →  HOLDS
  "x = (x * y) * z"  →  LHS bare x.  RHS uses {x,y,z}.  {x}≠{x,y,z}  →  FAILS
  "x = x * x"  →  LHS bare x.  RHS uses only {x}.  {x}={x}  →  HOLDS
  "x * y = (y * z) * w"  →  L={x,y}  R={y,z,w}  →  different  →  FAILS

═══ PROCEDURE ═══
Write down E1's left side and right side (split at the = sign).
Write down E2's left side and right side.
Work through the 4 tests in order: LP → RP → C0 → VARS.
For each, state E1's result then E2's result.
If you find E1=HOLD and E2=FAIL → stop, answer FALSE, cite that test.
If all 4 tests checked and no separation found → answer TRUE.

═══ WORKED EXAMPLE ═══
E1: x * y = (y * x) * x        E2: x = (x * y) * z

E1 left = "x * y"              E1 right = "(y * x) * x"
E2 left = "x"                  E2 right = "(x * y) * z"

LP: E1 first(L)=x first(R)=y → x≠y → FAIL
    E2 first(L)=x first(R)=x → x=x → HOLD
    Not a separation (separation needs E1=HOLD).

RP: E1 last(L)=y  last(R)=x → y≠x → FAIL
    E2 last(L)=x  last(R)=z → x≠z → FAIL
    Both FAIL → no separation.

C0: E1 both sides have * → HOLD
    E2 LHS is bare x, RHS has * → FAIL
    E1=HOLD, E2=FAIL → SEPARATION → answer FALSE.

VERDICT: FALSE
REASONING: C0 separates E1 from E2.
PROOF: E1 holds on C0 (both sides contain *). E2 fails (LHS is bare x, RHS has *).
COUNTEREXAMPLE: Witness C0. E1 holds. E2 fails because its left side is a lone variable while its right side is a product.

═══ REMINDERS ═══
• These are TEXT-INSPECTION checks — no arithmetic, no Cayley tables.
• LP / RP: just find the first / last letter on each side of =.
• C0: just look for the * symbol on each side.
• VARS: list the distinct letters on each side; compare sets.
• Each test looks WITHIN one equation (left side vs right side).
  Never compare properties across E1 and E2; only compare their HOLD/FAIL results.
• The default answer is TRUE. Say FALSE only when a specific test clearly shows
  E1=HOLD and E2=FAIL.
