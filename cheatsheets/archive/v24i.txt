═══ QUICK TRUE ═══

═══ STEP 0: SPLIT ON = ═══
Write out the four pieces:

"x = (big * expr)" → left side is just "x", a bare variable with NO *.

═══ 4 STRUCTURAL TESTS — find E1 result and E2 result, then look up the table ═══

TEST 1 — LP: first letter (a–z) on left side vs first letter on right side.
  Same → HOLD.  Different → FAIL.
  Tip: scan left-to-right past ALL opening parentheses to find the first letter.
  "(y * x)" → y.  "((y * x) * (x * y)) * z" → y.  "(x * x) * y" → x.

TEST 2 — RP: last letter on left side vs last letter on right side.
  Same → HOLD.  Different → FAIL.
  Tip: skip all trailing ) to find the last letter.
  "(x * y)" → y.  "(y * (x * y))" → y.  "((y * x) * x) * (y * x)" → x.

TEST 3 — C0: does left side contain *?  Does right side contain *?
  Both have * → HOLD.  Neither has * and same variable → HOLD.  Otherwise → FAIL.
  "x" has no *. "x * y" has *. Check each side independently in each equation.
  C0 checks ONLY whether * is present. Do not consider which variables appear.

TEST 4 — VARS: set of distinct letters on left vs set on right.
  Both have *: same set → HOLD, different set → FAIL.
  One side is bare variable v, other has *: HOLD only if * side uses only v.
  Neither has * and same variable → HOLD.  Otherwise → FAIL.

═══ DECISION TABLE — use after EVERY structural test ═══

  E1=HOLD  E2=HOLD  →  no separation → next test
  E1=HOLD  E2=FAIL  →  ★ SEPARATION → answer FALSE ★
  E1=FAIL  E2=HOLD  →  no separation → next test
  E1=FAIL  E2=FAIL  →  no separation → next test

ONLY the second row is separation. Read the row carefully every time.
After all 4 structural tests with no separation → go to RESCUE CHECKS below.

═══ RESCUE 1 — T3R MAGMA CHECK ═══
Run ONLY if: TEST 2 (RP) showed E1=HOLD, and no structural test found separation.
(If RP showed E1=FAIL, skip this rescue entirely.)

T3R rule: a ★ b = next(b). The LEFT input does NOT matter.
  next(0) = 1    next(1) = 2    next(2) = 0

Assign values to variables in order of first appearance across both equations:
  first variable = 0, second = 1, third = 2, fourth = 0, fifth = 1.

Evaluate EACH SIDE of E1 from innermost ★ outward:
  1. Find an innermost a ★ b (both a, b are already numbers or assigned variables).
  2. Look up b's numeric value. Result = next(b). Ignore a.
  3. Replace that whole a ★ b with the result number.
  4. Repeat until only a number remains.
  5. That number is the side's value.

Then evaluate each side of E2 the same way.

  E1 left = E1 right  AND  E2 left ≠ E2 right  →  ★ SEPARATION → FALSE ★
  Anything else → no separation → continue to Rescue 2.

═══ RESCUE 2 — T3L MAGMA CHECK ═══
Run ONLY if: TEST 1 (LP) showed E1=HOLD, and nothing above found separation.
(If LP showed E1=FAIL, skip this rescue entirely.)

T3L rule: a ★ b = next(a). The RIGHT input does NOT matter.
Same next function. Same variable assignment.

Evaluate each side of E1 and E2 from innermost outward, but now:
  For each a ★ b: look up a's numeric value. Result = next(a). Ignore b.

  E1 left = E1 right  AND  E2 left ≠ E2 right  →  ★ SEPARATION → FALSE ★
  Otherwise → no separation → answer TRUE.
