E1 is {{ equation1 }} and E2 is {{ equation2 }}

TASK: Does E1 imply E2 over all magmas?
DEFAULT: TRUE. Say FALSE only when a test below proves separation.

═══ QUICK TRUE ═══
If E1 is "x = T" and x does NOT appear anywhere in T → answer TRUE.

═══ STEP 0: SPLIT ON = ═══
Write out the four pieces first:
  E1_L = (before = in E1)    E1_R = (after = in E1)
  E2_L = (before = in E2)    E2_R = (after = in E2)
"x = (big * expr)" → left side is just "x", a bare variable with NO *.

═══ 4 TESTS — for each, find E1 result and E2 result, then use the decision table ═══

TEST 1 — LP: first letter (a–z) on left side vs first letter on right side.
  Same → HOLD.  Different → FAIL.

TEST 2 — RP: last letter on left side vs last letter on right side.
  Same → HOLD.  Different → FAIL.
  Tip: scan right-to-left past all ) to find the last letter.

TEST 3 — C0: does left side contain *? Does right side contain *?
  Both have * → HOLD.  Neither has * and same variable → HOLD.  Otherwise → FAIL.
  Note: "x = T" → left side is just "x" → NO * → always FAILS.

TEST 4 — VARS: set of distinct letters on left vs set on right.
  Both have *: same set → HOLD, different set → FAIL.
  One side is bare variable v, other has *: HOLD only if * side uses only v.
  Neither has * and same variable → HOLD.  Otherwise → FAIL.

═══ DECISION TABLE (use this EVERY time) ═══
After computing E1 and E2 results for a test, look up the answer here:

  E1=HOLD  E2=HOLD  →  no separation, next test
  E1=HOLD  E2=FAIL  →  ★ SEPARATION FOUND → answer FALSE ★
  E1=FAIL  E2=HOLD  →  no separation, next test
  E1=FAIL  E2=FAIL  →  no separation, next test

Only the second row (E1=HOLD + E2=FAIL) is a separation.
After all 4 tests with no separation → answer TRUE.

═══ OUTPUT FORMAT ═══
Do the full analysis first. Write the verdict block LAST.
  VERDICT: TRUE or FALSE
  REASONING: (one sentence)
  PROOF: (brief)
  COUNTEREXAMPLE: (if FALSE: name the test; if TRUE: "None")

HARD RULE: Your VERDICT must match your analysis.
If you found E1=HOLD and E2=FAIL on any test → VERDICT: FALSE.
If you found no such pair on any test → VERDICT: TRUE.
Do not override your own analysis.

═══ EXAMPLE A — FALSE via C0 ═══
E1: x * y = (y * x) * x     E2: x = (x * y) * z
E1_L="x * y"  E1_R="(y * x) * x"  |  E2_L="x"  E2_R="(x * y) * z"

LP:  E1 first(L)=x first(R)=y → FAIL | E2 first(L)=x first(R)=x → HOLD
     Table: E1=FAIL E2=HOLD → no separation
RP:  E1 last(L)=y last(R)=x → FAIL | E2 last(L)=x last(R)=z → FAIL
     Table: E1=FAIL E2=FAIL → no separation
C0:  E1 L has *, R has * → HOLD | E2 L="x" no * → FAIL
     Table: E1=HOLD E2=FAIL → ★ SEPARATION ★

VERDICT: FALSE
REASONING: C0 separates. E1 holds, E2 fails.
PROOF: E1 left "x * y" has *, right has *. E2 left "x" has no *.
COUNTEREXAMPLE: Witness C0.

═══ EXAMPLE B — FALSE via LP ═══
E1: x = x * (y * z)     E2: (x * x) * y = (y * x) * z
E1_L="x"  E1_R="x * (y * z)"  |  E2_L="(x * x) * y"  E2_R="(y * x) * z"

LP:  E1 first(L)=x first(R)=x → HOLD | E2 first(L)=x first(R)=y → FAIL
     Table: E1=HOLD E2=FAIL → ★ SEPARATION ★

VERDICT: FALSE
REASONING: LP separates. E1 first letters match, E2's differ.
PROOF: E1: x=x. E2: x≠y.
COUNTEREXAMPLE: Witness LP.

═══ EXAMPLE C — TRUE ═══
E1: x = (y * x) * (z * (w * y))     E2: x = ((y * x) * x) * x
E1_L="x"  E1_R="(y * x) * (z * (w * y))"  |  E2_L="x"  E2_R="((y * x) * x) * x"

LP:  E1 first(L)=x first(R)=y → FAIL | E2 first(L)=x first(R)=y → FAIL
     Table: E1=FAIL E2=FAIL → no separation
RP:  E1 last(L)=x last(R)=y → FAIL | E2 last(L)=x last(R)=x → HOLD
     Table: E1=FAIL E2=HOLD → no separation
C0:  E1 L="x" no * → FAIL | E2 L="x" no * → FAIL
     Table: E1=FAIL E2=FAIL → no separation
VARS: E1 L={x} R={x,y,z,w} → FAIL | E2 L={x} R={x,y} → FAIL
     Table: E1=FAIL E2=FAIL → no separation
No separation on any test.

VERDICT: TRUE
REASONING: No test separates E1 from E2.
PROOF: All 4 tests checked; none had E1=HOLD with E2=FAIL.
COUNTEREXAMPLE: None
