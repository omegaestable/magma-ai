E1 is {{ equation1 }} and E2 is {{ equation2 }}

TASK: Does E1 imply E2 over all magmas?
DEFAULT: TRUE. Say FALSE only when a test below proves separation.

═══ QUICK TRUE ═══
If E1 is literally identical to E2 (ignoring whitespace) → answer TRUE.

═══ STEP 0: SPLIT ON = ═══
Write out the four pieces:
  E1_L = (before = in E1)    E1_R = (after = in E1)
  E2_L = (before = in E2)    E2_R = (after = in E2)
"x = (big * expr)" → left side is just "x", a bare variable with NO *.

═══ 4 STRUCTURAL TESTS — find E1 result and E2 result, then look up the table ═══

TEST 1 — LP: first letter (a–z) on left side vs first letter on right side.
  Same → HOLD.  Different → FAIL.
  Tip: scan left-to-right past ALL opening parentheses to find the first letter.
  "(y * x)" → y.  "((y * x) * (x * y)) * z" → y.  "(x * x) * y" → x.

TEST 2 — RP: last letter on left side vs last letter on right side.
  Same → HOLD.  Different → FAIL.
  Tip: skip all trailing ) to find the last letter.
  "(x * y)" → y.  "(y * (x * y))" → y.  "((y * x) * x) * (y * x)" → x.

TEST 3 — C0: does left side contain *?  Does right side contain *?
  Both have * → HOLD.  Neither has * and same variable → HOLD.  Otherwise → FAIL.
  "x" has no *. "x * y" has *. Check each side independently in each equation.
  C0 checks ONLY whether * is present. Do not consider which variables appear.

TEST 4 — VARS: set of distinct letters on left vs set on right.
  Both have *: same set → HOLD, different set → FAIL.
  One side is bare variable v, other has *: HOLD only if * side uses only v.
  Neither has * and same variable → HOLD.  Otherwise → FAIL.

═══ DECISION TABLE — use after EVERY structural test ═══

  E1=HOLD  E2=HOLD  →  no separation → next test
  E1=HOLD  E2=FAIL  →  SEPARATION → answer FALSE
  E1=FAIL  E2=HOLD  →  no separation → next test
  E1=FAIL  E2=FAIL  →  no separation → next test

ONLY the second row is separation. Read the row carefully every time.
After all 4 structural tests with no separation → go to SPINE DEPTH CHECK below.

═══ SPINE DEPTH CHECK ═══
Run ONLY if LP showed E1=HOLD AND E2=HOLD, no structural separation found.

Left-spine depth: in the RHS, count opening ( at the start before the first
letter. Depth = count + 1. The first letter MUST be the LHS variable.
If different → not pure left-spine → skip → go to T3R RESCUE.

Both pure left-spine? E1 depth n, E2 depth m:
  n divides m → no separation → go to T3R RESCUE.
  n does NOT divide m → SEPARATION → answer FALSE.

═══ T3R RESCUE — ALGEBRAIC MAGMA CHECK ═══
Run ONLY if: RP showed E1=HOLD, and no earlier separation found.
(If RP showed E1=FAIL, skip rescue entirely → answer TRUE.)

T3R rule: a * b = next(b). The LEFT input does NOT matter. Only b matters.
  next(0) = 1    next(1) = 2    next(2) = 0

Step A — Assign numbers to variables by first appearance across both equations:
  1st distinct variable = 0, 2nd = 1, 3rd = 2, 4th = 0, 5th = 1.

Step B — REWRITE each side fully with numbers BEFORE evaluating.
  Replace every variable with its number. Work ONLY with numbers from here on.
  Example: "y * (z * x)" with x=0,y=1,z=2 becomes "1 * (2 * 0)"

Step C — Evaluate from innermost * outward:
  1. Find innermost a * b where both a and b are already numbers.
    If two innermost * exist, pick either — order does not matter.
  2. Result = next(b). Ignore a completely.
  3. Replace that (a * b) with the result number. Repeat until one number remains.

Step D — Compare E1 sides:
  CRITICAL: If E1 LHS ≠ E1 RHS → STOP. No separation possible. Answer TRUE.
  Only if E1 LHS = E1 RHS → continue and evaluate E2 the same way.
  E2 LHS = E2 RHS → no separation → answer TRUE.
  E2 LHS ≠ E2 RHS → SEPARATION → answer FALSE

═══ T3L — LEFT-SHIFT MAGMA ═══
If no separation yet, repeat Steps A–D using this rule instead:
  a * b = next(a). The RIGHT input does NOT matter. Only a matters.
Same abort: E1 LHS ≠ RHS → STOP → TRUE. Only E1=match + E2=mismatch → FALSE.

═══ OUTPUT FORMAT ═══
Write your analysis in plain text. Then write these four lines at the end:
  VERDICT: TRUE or FALSE
  REASONING: (one sentence)
  PROOF: (brief)
  COUNTEREXAMPLE: (if FALSE: name the test; if TRUE: "None")

HARD RULE: Your VERDICT line must match your analysis.
If ANY test or rescue had E1=HOLD/match and E2=FAIL/mismatch → VERDICT: FALSE.
If nothing separated → VERDICT: TRUE.

═══ EXAMPLE A — FALSE via C0 ═══
E1: x * y = (y * x) * x     E2: x = (x * y) * z
E1_L="x * y"  E1_R="(y * x) * x"  |  E2_L="x"  E2_R="(x * y) * z"

LP:  E1 first(L)=x first(R)=y → FAIL | E2 first(L)=x first(R)=x → HOLD
     → E1=FAIL E2=HOLD → no separation
RP:  E1 last(L)=y last(R)=x → FAIL | E2 last(L)=x last(R)=z → FAIL
     → E1=FAIL E2=FAIL → no separation
C0:  E1 L has *, R has * → HOLD | E2 L="x" no * → FAIL
  → E1=HOLD E2=FAIL → SEPARATION

VERDICT: FALSE
COUNTEREXAMPLE: C0.

E1: x = x * (y * z)     E2: (x * x) * y = (y * x) * z
E1_L="x"  E1_R="x * (y * z)"  |  E2_L="(x * x) * y"  E2_R="(y * x) * z"

  → E1=HOLD E2=FAIL → SEPARATION

COUNTEREXAMPLE: None

═══ RULES ═══
• Split each equation at = first. "x = stuff" → left is "x" (no *).
• Use the DECISION TABLE after every structural test.
• E1=HOLD + E2=FAIL = separation = FALSE. Everything else = continue.
• Default is TRUE. Say FALSE only with a clear separation.
• For LP: scan left-to-right past ALL opening parentheses to find the first letter.
• For RP: scan right-to-left past ALL closing parentheses to find the last letter.
• Flow: structural → spine (LP both HOLD) → T3R (RP E1=HOLD) → T3L → TRUE.
• T3R: a * b = next(b). Left argument ignored. Rewrite numbers FIRST.
• T3R abort: If E1 LHS ≠ RHS → STOP → TRUE. Only E1=match + E2=mismatch = FALSE.
• Write VERDICT last, it MUST match your analysis.
• Do NOT use ## headers, numbered steps, bullet lists, or \boxed{}.
• Use plain text analysis, then VERDICT/REASONING/PROOF/COUNTEREXAMPLE lines.
