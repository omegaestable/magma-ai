WITNESS RULES:
1. LP witness: x*y = x.
- For x = T, LP is valid only when the leftmost leaf of T is literally x.
- For non-x = T equations, LP is valid only when whole-side rewriting makes E1 a literal identity such as x = x or z = z.
- If LP rewrites E1 to x = y, x = z, z = w, or any non-identity variable equation, discard LP.

2. RP witness: x*y = y.
- For x = T, RP is valid only when the rightmost leaf of T is literally x.
- For non-x = T equations, RP is valid only when whole-side rewriting makes E1 a literal identity such as x = x or z = z.
- If RP rewrites E1 to x = y, x = z, z = w, or any non-identity variable equation, discard RP.
- If E1 passes RP and E2 is x = U with rightmost leaf not x, RP refutes.

3. C0 witness: x*y = 0.
- Every composite product becomes 0, but a bare variable stays itself.
- HARD LOCK: C0 is usable only if the full C0 rewrite makes E1 a literal identity such as 0 = 0 or x = x.
