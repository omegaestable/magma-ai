MAGMA IMPLICATION DECISION GUIDE

Given E1 and E2 (equations over magmas with operation *):
Does every magma satisfying E1 also satisfy E2?

=== ANSWER TRUE ONLY IN THESE CASES ===

CASE 1: One side of E1 is a single variable NOT on the other side.
  Example: E1 = "x = y * (z * w)" → x not on right → TRUE for any E2.
  But: E1 = "x = y * (x * z)" → x IS on right → not this case.

CASE 2: E2 is "x = x" → always TRUE.

CASE 3 (LEFT PROJECTION): E1 has form "x = x * TERM"
  where TERM starts with a variable OTHER THAN x,
  and that variable appears only ONCE in TERM.
  This means E1 forces a*b = a (left projection).
  Then check E2: look at the FIRST letter on each side of E2.
  If same first letter on both sides → TRUE.
  If different first letters → FALSE.
  
  ✓ x = x*(y*z): TERM starts with y, y once → forces a*b=a
  ✓ x = x*(y*(x*z)): TERM starts with y, y once → forces a*b=a
  ✗ x = x*(x*y): TERM starts with x → does NOT force a*b=a

CASE 4 (RIGHT PROJECTION): E1 has form "x = TERM * x"
  where TERM starts with a variable OTHER THAN x,
  and that variable appears only ONCE in TERM.
  This means E1 forces a*b = b (right projection).
  Then check E2: look at the LAST letter on each side of E2.
  If same last letter on both sides → TRUE.
  If different last letters → FALSE.

  ✓ x = y*x → forces a*b=b
  ✓ x = (y*z)*x → forces a*b=b
  ✗ x = ((x*y)*z)*x: TERM starts with x → NOT this case

=== ANSWER FALSE OTHERWISE ===

Extra FALSE evidence (counterexample magmas):
- If both sides of E1 start with the same variable but E2's sides
  start with DIFFERENT variables → FALSE (left projection refutes).
- If both sides of E1 end with the same variable but E2's sides
  end with DIFFERENT variables → FALSE (right projection refutes).

DEFAULT: If none of the TRUE cases above apply → FALSE.
Most equational implications are FALSE. When uncertain, say FALSE.
