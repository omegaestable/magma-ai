MAGMA IMPLICATION CHEATSHEET

Read * as the binary operation. Use this order:
instant -> singleton -> substitution -> duality -> multiplicity check -> rewrite -> stock magmas -> linear magmas -> soft heuristics.

1. INSTANT
- Eq1 = Eq2: TRUE.
- Eq2 is x=x: TRUE.
- Eq1 is x=x and Eq2 is not: FALSE.

2. SINGLETON / E2-TYPE
- x=y implies every law.
- More generally, if LHS-vars and RHS-vars are disjoint and at least one side uses *, treat Eq as singleton-forcing: it behaves like x=y.
- Therefore: if Eq1 is singleton-forcing, TRUE. If Eq2 is singleton-forcing and Eq1 is not, usually FALSE.
- High-yield pattern: x=t(vars-not-x) or t(vars-not-x)=x with at least one * strongly suggests singleton-forcing.

3. DIRECT TRUE RULES
- If Eq2 is a substitution instance of Eq1, TRUE immediately.
- Also check duality: reverse every product in both equations; the answer is unchanged. If dual(Eq2) is a substitution instance of Eq1, count that as strong TRUE evidence.
- Projection laws often prove nested targets by repeated collapse.
  left projection: x*y=x.
  right projection: x*y=y.
  Nested variants with the same collapse direction are often TRUE.

4. BALANCED-VS-UNBALANCED REFUTATION
- For each variable v, compare occ_LHS(v) and occ_RHS(v).
- If Eq1 is balanced for every variable but Eq2 is not, FALSE.
- Reason: rewriting under a balanced law preserves multiplicity balance.

5. REWRITE-CHAIN TRUE RULE
- If Eq1 and Eq2 use the same variables and Eq2 looks like a rearranged/simplified version of Eq1, try a short rewrite chain before guessing FALSE.
- Many hard TRUEs are not obvious from surface form but collapse by a few rewrites.

6. FIRST FALSE PASS: STOCK SMALL MAGMAS
- If Eq1 holds in one of these and Eq2 fails there, FALSE.
- const0 00/00, const1 11/11.
- left-zero 00/11, right-zero 01/01.
- xor 01/10.
- and 00/01, or 01/11.
- These tiny tables refute many hard negatives.

7. SECOND FALSE PASS: SMALL STRUCTURED MAGMAS
- Try linear magmas x*y = a x + b y mod p for small primes, especially p=3.
- Useful size-3 tables:
  012/201/120 = 2x+y mod 3.
  021/102/210 = x+2y mod 3.
  012/120/201 = z3 addition.
  000/111/222 = left projection on 3.
  012/012/012 = right projection on 3.

8. SOFT HEURISTICS ONLY
- If Eq2 introduces a variable absent from Eq1, FALSE is more plausible.
- If Eq1 is syntactically stronger (more ops/depth, same variable support), TRUE is more plausible.
- These are priorities, not proofs.

9. DISCIPLINE
- A real TRUE should come from singleton forcing, substitution, dual-substitution, or an actual rewrite argument.
- A real FALSE should come from multiplicity refutation or a concrete magma.
- Do not answer FALSE only because a quick search failed.
