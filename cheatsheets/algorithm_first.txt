MAGMA IMPLICATION CHEATSHEET

Read * as the binary operation in the given equations.
Use this order: singleton check -> specialization -> size-2 magma -> linear magma -> heuristics.

1. INSTANT CASES
- Eq1 and Eq2 identical: TRUE.
- Eq2 is x=x: TRUE.
- Eq1 is x=x and Eq2 is not: FALSE.

2. SINGLETON / E2-CLASS (highest coverage)
- If Eq1 forces every magma to have one element, TRUE.
- If Eq2 forces singleton and Eq1 clearly does not, FALSE.
- Safe singleton triggers:
  x=y.
  LHS-vars and RHS-vars are disjoint and at least one side uses *.
- Many x=t or t=x laws with extra variables are also E2-equivalent; exact offline class size is 1496/4694.

3. SPECIALIZATION
- If Eq2 is obtained from Eq1 by variable substitution only, TRUE.
- Example pattern: x=x*y -> x=x*x by y:=x.

4. MATCHING-INVARIANT REFUTATION
- For each variable v compute d(v)=occ_LHS(v)-occ_RHS(v).
- If every d(v)=0 in Eq1 but some d(v)!=0 in Eq2, FALSE.
- Intuition: rewriting under a multiplicity-balanced law cannot create imbalance.

5. SIZE-2 COUNTEREXAMPLE PASS (highest FALSE yield)
- If Eq1 holds in one table below and Eq2 fails there, FALSE.
- 00/00 const0: comm assoc.
- 11/11 const1: comm assoc.
- 00/11 left-zero: idem left0 assoc.
- 01/01 right-zero: idem right0 assoc.
- 01/10 xor: comm assoc.
- 00/01 and: idem comm assoc.
- 01/11 or: idem comm assoc.
- 11/10 nand: comm only.
- 11/01 implication-like: noncomm nonassoc.
- 01/00 left-and-not: noncomm nonassoc.
- 00/10 right-xor-and: noncomm nonassoc.

6. LINEAR / SMALL STRUCTURED MAGMAS
- Next try x*y=(a x+b y) mod p for p in {2,3,5,7}.
- Highest-yield ready-made size-3 tables:
  012/201/120 = 2x+y mod 3.
  021/102/210 = x+2y mod 3.
  012/120/201 = z3-add (comm assoc).
  000/111/222 = left projection on 3.
  012/012/012 = right projection on 3.

7. STRUCTURAL HEURISTICS
- If Eq2 uses a variable absent from Eq1, usually FALSE.
- If Eq1 has more variables, more operations, or greater depth than Eq2, TRUE is more plausible but not proved.
- Duality: reverse every product in both equations; the answer must stay the same.

8. WHAT COUNTS AS A REAL TRUE
- instant case
- singleton forcing
- direct specialization
- explicit rewrite chain to Eq2 or to a common normal form

9. WHAT COUNTS AS A REAL FALSE
- multiplicity refutation
- explicit finite magma
- linear/translation-invariant counterexample

10. DEFAULT POSTURE
- Small counterexamples are common; check them before inventing proofs.
- Failing to find a small counterexample is not a proof.
