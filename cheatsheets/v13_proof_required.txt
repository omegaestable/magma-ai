LP (Left Projection): x*y = x for all x,y.

RP (Right Projection): x*y = y for all x,y.
  Table (size 2): [[0,1],[0,1]]

  LEAF variable in the parse tree (ignore all left-side arguments).

=== KEY PROOF STRATEGIES FOR TRUE ===

Strategy 1 — ABSORPTION: If {{ equation1 }} forces x*y = x (left-absorb) or x*y = y (right-
absorb), then ALL terms collapse to a single variable. {{ equation1 }} then implies almost
any {{ equation2 }}. Check: substitute specific values into {{ equation1 }} to derive x*y=x or x*y=y.

Strategy 2 — VARIABLE SUBSTITUTION: Set free variables equal.
  E.g., from "x = x * (y * (z * (y * y)))", set y=x and z=x to get
  x = x * (x * (x * (x * x))). Then show this forces x*y=x for all y.

Strategy 3 — TRANSITIVITY: If {{ equation1 }} implies some intermediate law L, and L implies
{{ equation2 }}, then {{ equation1 }} implies {{ equation2 }}. Common intermediate: left-absorb, right-absorb,
idempotency (x*x=x), commutativity (x*y=y*x).

Strategy 4 — TERM REWRITING: If {{ equation1 }} says x = f(x,y,...), substitute this
identity into {{ equation2 }} systematically. If every occurrence of "x" in {{ equation2 }} can be
replaced using {{ equation1 }} to make {{ equation2 }} hold, then TRUE.

=== FALSE COUNTEREXAMPLE COOKBOOK ===

When {{ equation1 }} is a "weak" equation (LP or RP satisfies it), use that magma as CE:

LP satisfies these common forms:
  x = x * (... anything ...) — because x*(...)=x on LP
  x = (... x-rooted term ...) * (... anything ...) — if inner term has x as leftmost
  Example: "x = x * ((y * z) * (x * w))" → LP satisfies, RHS=x*(y*x)=x*x=x ✓

RP satisfies:
  x = y * x — right-absorb law
  x = (... anything ...) * x — because (...)*x = x on RP

TO DISPROVE {{ equation1 }} → {{ equation2 }} (i.e., show FALSE):
  1. Pick a candidate CE magma (LP, RP, XOR, Z3A, C0)
  2. Substitute the magma's operation into {{ equation1 }}. Check ALL variable combinations.
  3. If {{ equation1 }} holds on all inputs → magma satisfies {{ equation1 }} ✓
  4. Find ONE input combination where {{ equation2 }} fails → magma refutes {{ equation2 }} ✓
  5. Present the magma and the failing assignment.

=== COMMON MISTAKES TO AVOID ===

MISTAKE 1: Using LP/RP as CE without checking leftmost/rightmost leaf first.
  "x = ((y*y)*z)*x": leftmost leaf of RHS = y, not x. LP gives x=y → FAILS {{ equation1 }}!
  "x = x*(y*(z*(x*y)))": rightmost leaf = y, not x. RP gives x=y → FAILS {{ equation1 }}!
  If leaf ≠ x, that magma CANNOT be CE. Do this check FIRST, before anything.

MISTAKE 2: Defaulting to FALSE when no CE is obvious.
  If you cannot find a valid small CE, keep checking or prove TRUE explicitly.

MISTAKE 4: Rejecting a CE as invalid and then still answering FALSE.
  Invalid CE means discard it; if no other CE works, answer TRUE.

MISTAKE 3: Confusing left-absorb (x*y=x) with right-absorb (x*y=y).
  These are different! LP gives x*y=x. RP gives x*y=y.
