EQUATIONAL IMPLICATION OVER MAGMAS ‚ÄĒ CHEATSHEET

A MAGMA (M,‚óá) is a set M with a binary operation ‚óá. No axioms are assumed.
"Eq1 IMPLIES Eq2" means: every magma satisfying Eq1 also satisfies Eq2.

‚ēź‚ēź‚ēź DECISION PROCEDURE ‚ēź‚ēź‚ēź

Step 1: CHECK TRIVIAL CASES
- If Eq1 = Eq2 (syntactically identical): TRUE
- If Eq2 is "x = x": TRUE (tautology, everything implies it)
- If Eq1 is "x = x": FALSE (unless Eq2 is also "x = x")
- If Eq1 is "x = y" or equivalent to it: TRUE (singleton law implies everything)
- If Eq2 is "x = y" or has form forcing M to be singleton: likely FALSE unless Eq1 also forces singleton

Step 2: TRY DIRECT PROOF (substitution/rewriting)
Given Eq1: LHS1 = RHS1. We can use this as a rewrite rule in both directions.
- Start from LHS2 of Eq2. Apply Eq1 rewrites (substitute equals for equals) anywhere in the term.
  If you can reach RHS2, then TRUE.
- Also try: start from both sides and see if they can be rewritten to a common term.
- Key: you can substitute any subterm matching one side of Eq1 with the other side.
  You can also substitute variables in Eq1 with arbitrary terms before applying.

Step 3: TRY COUNTEREXAMPLE (small magma)
Build a multiplication table for ‚óá on a small set {0,1} or {0,1,2}.
Check: does this magma satisfy Eq1 but violate Eq2?
If yes: FALSE.

Size-2 magma tables (values of a‚óáb for a,b ‚ąą {0,1}):
  ‚óá|0 1    Specify as [row0, row1] where row_i = [i‚óá0, i‚óá1]
  0|a b    There are 16 possible 2-element magmas.
  1|c d

Useful size-2 magmas for counterexamples:
  Left-zero:  [[0,0],[1,1]]  (x‚óáy=x, satisfies Eq25: x=(x‚óáy)‚óáx)
  Right-zero: [[0,1],[0,1]]  (x‚óáy=y)
  Constant-0: [[0,0],[0,0]]  (x‚óáy=0, satisfies any "constant" law)
  Constant-1: [[1,1],[1,1]]  (x‚óáy=1)
  AND:        [[0,0],[0,1]]  (idempotent, commutative)
  OR:         [[0,1],[1,1]]  (idempotent, commutative)
  XOR:        [[0,1],[1,0]]  (commutative, x‚óáx=0, not idempotent)
  NAND:       [[1,1],[1,0]]
  Left-const: [[0,0],[1,1]] = Left-zero (listed above)
  Proj-swap:  [[1,0],[1,0]]  (x‚óáy = 1-y, constant on second arg)

Size-3 useful magmas (mod 3 arithmetic):
  x‚óáy = x+1 mod 3:        [[1,1,1],[2,2,2],[0,0,0]]
  x‚óáy = x+y mod 3:         addition group (commutative, associative)
  x‚óáy = 2x+2y mod 3:       satisfies many laws
  x‚óáy = x+y+1 mod 3:       shifted addition

LINEAR MAGMA METHOD: Try x‚óáy = ax+by+c (mod p) for small primes p.
Check if Eq1 is satisfied (substitute and verify mod p).
Then check if Eq2 is violated. This finds many counterexamples efficiently.

‚ēź‚ēź‚ēź KEY STRUCTURAL RULES ‚ēź‚ēź‚ēź

VARIABLE RULE: Count distinct variables on each side of each equation.
- Eq1 has vars V1, Eq2 has vars V2.
- If a variable appears in Eq2 but NOT in Eq1 (as a whole equation), 
  Eq1 may not constrain it ‚Üí evidence for FALSE (but not conclusive).

OPERATION COUNT: Let ops(E) = number of ‚óá operations in equation E.
- Equations with more operations are generally more specific/constraining.
- A more constraining equation (more ops in Eq1) is more likely to imply a less constraining one.

DEPTH RULE: Deeper nesting = more specific constraint. If Eq1 is deeper than Eq2, slight evidence for TRUE.

DUALITY: The DUAL of an equation swaps left and right arguments of every ‚óá.
  dual(x) = x, dual(a‚óáb) = dual(b)‚óádual(a)
  Example: x‚óá(y‚óáz) has dual (z‚óáy)‚óáx
  If Eq1 implies Eq2, then dual(Eq1) implies dual(Eq2).
  Some equations are self-dual. Dual pairs behave symmetrically.

‚ēź‚ēź‚ēź LANDMARK EQUATIONS ‚ēź‚ēź‚ēź

Eq1:   x = x                     [tautology, implied by everything, implies nothing]
Eq2:   x = y                     [singleton law, implies everything, ~1495 laws equivalent]
Eq3:   x = x‚óáx                  [idempotent fixpoint, 39 implications]
Eq4:   x = x‚óáy                  [left absorbing, implies Eq3]
Eq5:   x = y‚óáx                  [right absorbing, dual of Eq4]
Eq6:   x = y‚óáy                  [equivalent to Eq2 since x=y‚óáy and y=x‚óáx gives x=y]
Eq7:   x = y‚óáz                  [equivalent to Eq2]

Eq38:  x‚óáx = x‚óáy               [left-cancellation variant, equivalent to Eq4]
Eq39:  x‚óáx = y‚óáx               [right-cancel variant, equivalent to Eq5]
Eq40:  x‚óáx = y‚óáy               [all squares equal, implies Eq3 via x=x‚óáx]
Eq41:  x‚óáx = y‚óáz               [equivalent to Eq2]
Eq42:  x‚óáy = x‚óáz               [left-cancellation-like]

SINGLETON EQUIVALENCE: ~1495 equations are equivalent to Eq2 (x=y).
  Any equation with form "s = t" where s,t have disjoint variables (no variable in common)
  and at least one has a ‚óá operation forces M to be a singleton. Examples: Eq6, Eq7, Eq41, Eq46.
  Also: Eq14 (x = y‚óá(x‚óáy)), Eq15, Eq17, Eq18, Eq20, Eq21, Eq22, Eq24, Eq26-37, etc.
  Quick test: if LHS is just "x" and RHS has a variable not equal to x with no x in RHS ‚Üí singleton.

‚ēź‚ēź‚ēź COMMON IMPLICATION PATTERNS ‚ēź‚ēź‚ēź

KNOWN TRUE IMPLICATIONS:
- Eq4 (x=x‚óáy) ‚Üí Eq3 (x=x‚óáx): substitute y=x in Eq4.
- Eq4 (x=x‚óáy) ‚Üí Eq38 (x‚óáx=x‚óáy): from x=x‚óáy, substitute to get x‚óáx=(x‚óáy)‚óáx=x‚óáx.
- Anything equivalent to Eq2 ‚Üí everything.
- Eq43 (commutativity) ‚Üí does NOT imply Eq4512 (associativity), and vice versa.
- Eq4512 (associativity) ‚Üí does NOT imply Eq43 (commutativity).

SPECIALIZATION: If Eq2's equation is obtainable from Eq1 by substituting variables
  (e.g., replacing y with x), then Eq1 implies Eq2. This is the most common proof method.
  Example: "x‚óáy = y‚óáx" with y‚Üíx gives "x‚óáx = x‚óáx" = tautology.

COMPOSITION: If Eq1 implies EqA and EqA implies Eq2, then Eq1 implies Eq2 (transitivity).

‚ēź‚ēź‚ēź COMMON FALSE PATTERNS ‚ēź‚ēź‚ēź

INDEPENDENCE: Most pairs of "interesting" equations are independent (neither implies the other).
- Commutativity ‚ä¨ Associativity and Associativity ‚ä¨ Commutativity.
- Idempotency (Eq3) ‚ä¨ Commutativity (Eq43).
- Commutativity (Eq43) ‚ä¨ Idempotency (Eq3).
  Counterexample: Z with x‚óáy = x+y is commutative but not idempotent (x+x‚Č†x for x‚Č†0).

BASE RATE: The overall implication matrix is sparse. Most ordered pairs are FALSE.
  Only ~6% of all possible implications hold. If unsure, lean toward FALSE.

‚ēź‚ēź‚ēź PROOF CONSTRUCTION TIPS ‚ēź‚ēź‚ēź

FOR TRUE: Show a chain of equalities from LHS2 to RHS2 using Eq1 as rewrite rule.
  Each step: identify a subterm matching one side of Eq1, replace with the other side.
  You may instantiate Eq1's variables with any terms.
  Example proof that x‚óáy=y‚óáx implies (x‚óáy)‚óáz=(z‚óáy)‚óáx:
    (x‚óáy)‚óáz = (y‚óáx)‚óáz [apply comm to x‚óáy]... continue if possible.

FOR FALSE: Exhibit a specific magma table satisfying Eq1 but not Eq2.
  Verify by checking ALL assignments of variables (for size 2: only 4 or 8 checks).
  State: "The magma ({0,1}, ‚óá) with table [a‚óáb values] satisfies Eq1 because... 
  but violates Eq2 because for x=_, y=_: LHS=_ ‚Č† RHS=_"

‚ēź‚ēź‚ēź SYSTEMATIC COUNTEREXAMPLE SEARCH ‚ēź‚ēź‚ēź

For size-2 magmas, check all 16 tables. For each:
  1. Verify Eq1 holds for ALL variable assignments
  2. Find ANY assignment where Eq2 fails
  
Common counterexample magmas:
  [[0,0],[0,0]] satisfies: all "constant" equations, absorption laws
  [[0,1],[0,1]] satisfies: x‚óáy=y (right projection), associativity, Eq5
  [[0,0],[1,1]] satisfies: x‚óáy=x (left projection), associativity, Eq4
  [[0,1],[1,0]] satisfies: commutativity, x‚óá(x‚óáy)=y, (x‚óáy)‚óáy=x
  [[0,0],[1,0]] satisfies: idempotent, x‚óá(x‚óáy)=x‚óáy
  [[1,0],[0,1]] satisfies: commutativity, x‚óáx=1 (constant squares)

‚ēź‚ēź‚ēź ADVANCED: EQUIVALENCE CLASSES ‚ēź‚ēź‚ēź

Many equations form equivalence classes (they mutually imply each other).
The largest class is the "singleton" class (~1495 equations equivalent to Eq2).
Other notable classes:
- {Eq4, Eq38}: x=x‚óáy ‚ÜĒ x‚óáx=x‚óáy
- {Eq5, Eq39}: x=y‚óáx ‚ÜĒ x‚óáx=y‚óáx (dual of above)
- {Eq40}: x‚óáx=y‚óáy stands alone (weaker than Eq4/Eq5 but not singleton)
- {Eq43}: commutativity stands alone

‚ēź‚ēź‚ēź QUICK DECISION FLOWCHART ‚ēź‚ēź‚ēź

1. Are Eq1 and Eq2 identical? ‚Üí TRUE
2. Is Eq2 "x = x"? ‚Üí TRUE  
3. Is Eq1 "x = x"? ‚Üí FALSE
4. Is Eq1 equivalent to "x = y"? ‚Üí TRUE (check: disjoint vars forcing singleton)
5. Can you get Eq2 by substituting variables in Eq1? ‚Üí TRUE  
6. Can you rewrite LHS2‚ÜíRHS2 using Eq1? ‚Üí TRUE
7. Does a size-2 magma satisfy Eq1 but not Eq2? ‚Üí FALSE
8. Does a linear magma (ax+by+c mod p) satisfy Eq1 but not Eq2? ‚Üí FALSE
9. Does Eq2 introduce variables absent from Eq1? ‚Üí likely FALSE
10. Default: lean FALSE (~94% of pairs are FALSE)
